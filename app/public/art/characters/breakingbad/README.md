# Character images — breakingbad

Drop a character photo here named by its slug, with any of these extensions:
`.jpg`, `.jpeg`, `.png`, `.webp`, `.avif`.

The closest-match result shows this image; characters without a file fall back to a
themed initials avatar. Slug must match the character name exactly (see list).

| Character | Expected filename |
|---|---|
| Walter White | `walter-white.jpg` |
| Skyler White | `skyler-white.jpg` |
| Jesse Pinkman | `jesse-pinkman.jpg` |
| Hank Schrader | `hank-schrader.jpg` |
| Marie Schrader | `marie-schrader.jpg` |
| Walter White Jr. | `walter-white-jr.jpg` |
| Mike Ehrmantraut | `mike-ehrmantraut.jpg` |
| Saul Goodman | `saul-goodman.jpg` |
| Gus Fring | `gus-fring.jpg` |

_Note: these images are not shipped with the repo. Use only images you are licensed
to use, especially for a public deployment (Google-image results are typically
copyrighted studio stills)._
