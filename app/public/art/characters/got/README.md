# Character images — got

Drop a character photo here named by its slug, with any of these extensions:
`.jpg`, `.jpeg`, `.png`, `.webp`, `.avif`.

The closest-match result shows this image; characters without a file fall back to a
themed initials avatar. Slug must match the character name exactly (see list).
